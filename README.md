# Projeto-CV-Flask-Responsivo
Escopo MVC - Model View Control

O projeto consiste em uma página 100% responsiva de rolagem única, em formato de perfil profissional em modelo de Currículo. <br>
Todo o backend foi construído em Python - Flask, contando com um formulário de contato funcional. O fronte em Bootstrap, HTML e CSS com estilização de movimentos em Javascript.


Link da aplicação:<br>
<dvi class="row">
  <a href="https://mnsjr-cv-responsivo.herokuapp.com/" target="_blank">Acesse: mnsjr CV Responsivo</a>
</div>

Deploy feito no Heroku.
